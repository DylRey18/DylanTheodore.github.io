Biodiversity Image Project

I collaborated with three data scientist colleagues to develop an advanced image recognition machine learning model that analyzes the interactions between bees and flowers.

We created a comprehensive data pipeline in Python and Jupyter for extracting metadata and images from the ALA database, storing this information in a MySQL database using SQL queries. By integrating key data points, we enhanced pattern discovery, providing nuanced insights into biodiversity dynamics to support Australian scientists in their research.

Diving into the world of anime

I created an anime-themed data visualization dashboard using Tableau, featuring comprehensive insights into the historical growth of anime. Meat Consumption Worldwide

I developed a dynamic data visualization dashboard powered by Vega-Lite, presenting global meat consumption trends. Utilizing JSON format, the dashboard displays current and predicted future consumption patterns for poultry, pork, beef, and lamb. It offers a clear, interactive interface for exploring and understanding the evolving landscape of meat consumption globally
